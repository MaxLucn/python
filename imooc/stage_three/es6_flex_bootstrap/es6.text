函数、箭头函数
    箭头函数体内的 this 指向定义时所在的对象，不是实例化后的对象(在函数定义时绑定)
    什么是 this
        指函数执行过程中，自动生成的一个内部对象，是指当前的对象，名只在当前
        函数内部使用。
        运行是跟运行环境绑定
            浏览器： this === window
            Node 环境：this === global
        当函数被作为某个对象的方法调用的时候，this 指向那个对象
